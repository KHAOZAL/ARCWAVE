# ArcWave

ArcWave is a stablecoin-native DEX, bridge and pool intelligence prototype built for the Arc testnet.

## Features

- Stablecoin swap interface
- Route intelligence
- Liquidity add/remove interface
- Bridge interface
- Pool analytics
- WaveAgent AI-style assistant for slippage, pool health, liquidity depth and risk explanations

## Status

ArcWave is currently a testnet prototype.

Do not use real funds. This repository is intended for technical evaluation, grants, partnerships and ecosystem feedback.
