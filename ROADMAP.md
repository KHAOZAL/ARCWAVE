## Phase 3 — Analytics Layer

Status: In progress

Planned improvements:

- Total pool overview
- Selected pool analytics
- Swap count
- Volume tracking
- Liquidity tracking
- User activity tracking
- Pool health overview

## Phase 4 — ArcWave Points

Status: In progress

ArcWave Points will be used as an experimental reputation layer for identifying meaningful testnet participation.

Possible categories:

- Swap Tester
- Liquidity Provider
- Stablecoin Explorer
- Bug Hunter
- Governance Contributor
- Early Tester

Points are not a token and do not guarantee financial rewards.

## Phase 5 — Governance Experiments

Status: Planned

Possible governance tests:

- Vote on new pools
- Vote on pool fee settings
- Vote on liquidity incentives
- Vote on testnet campaigns
- Vote on UX priorities

## Phase 6 — Grant Expansion

Status: Planned

Grant support would help expand ArcWave into a broader stablecoin liquidity testing layer for Arc.

Potential milestones:

- Advanced analytics
- Better router
- Pool creation interface
- Leaderboard
- Public documentation
- Security improvements
- Contract upgrades
